# Telegram Bot for Notifications
Telegram-бот, который позволяет пользователям планировать уведомления. Пользователи могут отправить сообщение с указанием даты и времени, и бот напомнит им об этом в указанное время.

## Technologies
- Java 17
- Spring Boot
- Telegram Bot API
- JPA/Hibernate
- PostgreSQL
- Maven

## Usage

### Команды
- `/start`: Запустите бота и получите приветственное сообщение.
- `дд.ММ.гггг, ЧЧ:мм": Отправить сообщение с указанием даты и времени для отправки уведомления.

### Пример
- Отправить сообщение: "15.10.2024 14:30".
- Бот напомнит вам об этом в 14:30 15 октября 2024 года.
